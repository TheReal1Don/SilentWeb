# سايلنت ويب | SilentWeb

SilentWeb هو متصفح خفيف وسريع يركز على الخصوصية وتجربة التصفح النظيفة والبسيطة.

تم تطوير المتصفح باستخدام Microsoft Edge WebView2 بهدف توفير أداء سريع مع تقليل استهلاك الموارد والخدمات الخلفية، بالإضافة إلى حجب المتتبعات والإعلانات، ودعم DNS المشفر، وأنماط تصفح متعددة تناسب مختلف الاستخدامات.

مميزات المتصفح:

* سرعة وخفة عالية
* تركيز قوي على الخصوصية
* أوضاع متعددة (Silent / Stealth / Turbo / Normal)
* حجب الإعلانات والمتتبعات
* دعم DNS المشفر (DNS-over-HTTPS)
* جلسات تخفي مؤقتة ومعزولة
* واجهة بسيطة ونظيفة
* تقليل استهلاك الرام عبر تعليق التبويبات
* نظام مفضلة خفيف
* بحث ذكي
* مدير تحميلات بسيط وخفيف

تم تصميم SilentWeb ليقدم تجربة تصفح هادئة وسريعة بدون الميزات المزعجة أو الاستهلاك الزائد للموارد.

هذا المشروع تم تطويره لأغراض تعليمية وتجريبية وتعلم البرمجة.

# SilentWeb

SilentWeb is a lightweight minimal browser focused on speed, privacy, and a clean browsing experience.

Built using Microsoft Edge WebView2, SilentWeb aims to provide fast performance with reduced background activity, tracker blocking, DNS-over-HTTPS support, lightweight browsing modes, and isolated stealth sessions — all while keeping the browser simple and responsive.

Features:

* Lightweight and fast performance
* Privacy-focused browsing
* Silent / Stealth / Turbo / Normal modes
* Tracker and ad blocking
* DNS-over-HTTPS support
* Temporary isolated stealth sessions
* Minimal and clean interface
* Tab suspension for lower RAM usage
* Bookmark system
* Smart search support
* Lightweight download manager

SilentWeb is designed for users who want a quiet, minimal, and privacy-respecting browsing experience without unnecessary bloat.

This project is created for educational and learning purposes.
