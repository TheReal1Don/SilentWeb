المميزات الرئيسية (Features)
أداء فائق السرعة وخفة استثنائية: بُني باستخدام تقنية Microsoft Edge WebView2 لضمان استجابة سريعة جداً.

استهلاك منخفض للذاكرة (RAM): يتميز بنظام تعليق التبويبات (Tab Suspension) الذي يقوم بتعليق التبويبات غير النشطة تلقائياً للحفاظ على سلاسة أداء جهازك.

حماية متكاملة للخصوصية: حجب ذكي للمتتبعات والإعلانات مع دعم كامل لـ DNS المشفر (DNS-over-HTTPS).

أنماط تصفح مرنة:

Normal: للتصفح اليومي مع كامل الميزات.

Silent: تصفح بخصوصية تامة دون حفظ أي سجل أو ملفات تعريف ارتباط.

Turbo: لسرعة قصوى عبر حجب الصور، الخطوط، والوسائط الثقيلة.

Stealth: وضع التخفي الصارم مع توجيه الاتصال بالكامل عبر شبكة Tor.

أدوات مدمجة: نظام مفضلة ذكي، محرك بحث سريع، ومدير تحميلات خفيف صُمم ليعمل في الخلفية دون إثقال النظام.

فكرة المشروع (About)
وُلد SilentWeb ليكون البديل الأمثل للمتصفحات "الثقيلة" التي تستهلك موارد الجهاز بلا داعٍ. الفكرة قائمة على تقديم متصفح "نظيف" يضع السرعة كأولوية قصوى، تليها الخصوصية المطلقة. من خلال تسخير تقنية WebView2، تمكنت من بناء متصفح يقدم أداءً احترافياً بواجهة مستخدم بسيطة تبتعد عن التعقيد، مما يجعله الخيار المثالي للأجهزة ذات الموارد المحدودة أو للمستخدمين الذين يفضلون بيئة تصفح هادئة، سريعة، ومحميّة.

# SilentWeb

**A lightning-fast, lightweight browser designed for a seamless, minimal browsing experience.**

### Key Features

* **High Performance & Lightweight:** Built with Microsoft Edge WebView2 for exceptional speed and responsiveness.
* **Memory Efficiency:** Features an intelligent tab suspension system to keep your RAM usage minimal, even with multiple tabs open.
* **Privacy-First:** Native tracker and ad blocking, coupled with DNS-over-HTTPS support to keep your data secure.
* **Versatile Browsing Modes:** * **Normal:** For everyday use.
* **Silent:** Private browsing without history retention.
* **Turbo:** Optimized for speed by blocking images, fonts, and heavy media.
* **Stealth:** Maximum anonymity with integrated Tor routing and isolated sessions.


* **Built-in Tools:** Integrated lightweight download manager, smart search, and a streamlined bookmarking system.

### About

SilentWeb was born from the desire to create a "bloat-free" alternative to mainstream browsers. By leveraging WebView2, SilentWeb provides a professional browsing experience without the unnecessary background processes and resource overhead. This project is dedicated to users who appreciate a clean, quiet, and privacy-respecting environment.
